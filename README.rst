rc
==

I want to use same environment in my all computers.

.. sourcecode:: bash

   $ git clone git://github.com/sublee/rc.git
   $ ln -s rc/profile ~/.profile
   $ ln -s rc/vimrc ~/.vimrc
   $ ln -s rc/hgrc ~/.hgrc
   $ ln -s rc/matplotlibrc ~/.matplotlibrc
   $ sudo ln -s rc/limits.conf /etc/security/limits.conf
